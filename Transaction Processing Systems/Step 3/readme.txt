To build, type:
	tar -xvfz tpproject.tar.gz
	cd tpproject
	make
	./Test.exe
	
Test.exe is the test application.  It instantiates an instance of BANDB, our simplified database.  Then it performs the test actions described in the assignment to verify BANDB's ability to perform various actions.
